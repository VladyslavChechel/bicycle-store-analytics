| Атрибут         | Опис                                                           | Коментар                                       |
| --------------- | -------------------------------------------------------------- | ---------------------------------------------- |
| customer_key    | Ключ, що унікально ідентифікує кожен запис клієнта в таблиці   | “1,2,3,4…”                                     |
| customer_id     | Унікальний числовий ідентифікатор, присвоєний кожному клієнту  | “11000, 11001, 11002, 11003….”                 |
| customer_number | Буквено-цифровий ідентифікатор, що представляє клієнта         | “AW00011000, AW00011001,<br>AW00011002,......” |
| first_name      | Імʼя клієнта                                                   | “Elizabeth”                                    |
| last_name       | Прізвище клієнта                                               | “Johnson”                                      |
| country         | Країна проживання клієнта                                      | “Australia”                                    |
| marital_status  | Сімейний статус                                                | “Married”, “Single”                            |
| gender          | Стать                                                          | “Male”, “Female”, “Unknown”                    |
| birthdate       | Дата народження                                                | “1971-10-06”                                   |
| product_key     | Ключ, що унікально ідентифікує кожен запис продукту в таблиці  | “1,2,3,4…”                                     |
| product_id      | Унікальний ідентифікатор продукту для внутрішнього відстеження | “210, 211, 348….”                              |
| product_number  | Буквено-цифровий код, що представляє продукт                   | “FR-R92B-58”                                   |
| product_name    | Описова назва продукту, включаючи колір та розмір              | “Mountain-100 Black- 38”                       |
| category_id     | Унікальний ідентифікатор для категорії продукту                | “BI_MB”                                        |
| category        | Загальна класифікація продукту                                 | Bikes, Accessories, Clothing, Components       |
| sub_category    | Детальна класифікація продукту в межах категорії               | “Mountain Bikes”.....                          |
| product_line    | Лінійка продукції до якої належить товар                       | “Mountain”.....                                |
| maintenance     | Технічне обслуговування                                        | “Yes”, “No”                                    |
| cost            | Базова ціна продукту, $                                        | “1898”                                         |
| start_date      | Дата, коли продукт став доступним для продажу                  | “2011-07-01”                                   |
| order_number    | Унікальний ідентифікатор кожного замовлення                    | “SO43697”                                      |
| order_date      | Дата замовлення                                                | “2010-12-29”                                   |
| shipping_date   | Дата відправлення                                              | “2011-01-05”                                   |
| due_date        | Термін оплати                                                  | “2011-01-10”                                   |
| sales_amount    | Загальна сума, $                                               | “100.00”                                       |
| quantity        | Кількість одиниць товару                                       | “2”                                            |
| price           | Вартість одиниці товару, $                                     | “50”                                           |
